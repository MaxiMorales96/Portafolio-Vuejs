# Portafolio - Maximiliano A Morales (v2)

Instrucciones rápidas:

1. Instalar dependencias: `npm install`
2. Ejecutar en desarrollo: `npm run dev`
3. La foto está en `src/assets/photo.jpg`. Reemplazala si querés otra.
4. Los enlaces en el header y en cada proyecto son placeholders (poné tus links).

Diseño: fondo blanco, colores pasteles, responsive para celular/tablet/PC.
